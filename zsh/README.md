## Cheatsheet

### cli shortcuts

alt + w: delete all
ctrl + w: delete word
ctrl + a: line start
ctrl + d: EOF

### git shortcuts

| Alias | command |
|:---- |:--------------------------------|
| g    | git |
| ga   | git add |
| gc   | git commit -v |
| gc!  | git commit -v --amend |
| gca  | git commit -va |
| gca! | git commit -va --amend |
| gcm  | git checkout master |
| gco  | git checkout |
| ggl  | git pull origin $(current_branch) |
| gl   | git pull |
| glum | git pull upstream master |
| ggp  | git push origin $(current_branch) |
| gp   | git push |
| gr   | git remote |
| grv  | git remote -v |
| grb  | git rebase |
| grbm | git rebase master |
| grba | git rebase --abort |
| grbc | git rebase --continue |
| gss  | git status -s |
| gst  | git status |
